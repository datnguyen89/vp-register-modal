# vp-register-modal

## Install

```bash
npm install --save vp-register-modal
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'vp-register-modal'
import 'vp-register-modal/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
